  <files>.github/workflows/supabase-backup.yml, scripts/supabase-backup.sh</files>
  <action>
Create two files:

**scripts/supabase-backup.sh** — Bash script that:
1. Runs `pg_dump` against the Supabase PostgreSQL database using `DATABASE_URL` env var. Use `pg_dump --format=custom --no-owner --no-acl` for a compact, restorable dump. Output file named `backup-$(date -u +%Y%m%d-%H%M%S).dump`.
2. Uploads the dump to Cloudflare R2 using `aws s3 cp` with the S3-compatible endpoint. Target path: `s3://${R2_BUCKET_NAME}/supabase-backups/backup-YYYYMMDD-HHMMSS.dump`.
3. Lists all objects in the `supabase-backups/` prefix, sorted by date. If count > 10, deletes the oldest files until only 10 remain. Use `aws s3api list-objects-v2` + `aws s3 rm` for the rotation.
4. Prints summary: file uploaded, size, files deleted (if any), total backups remaining.
5. Uses `set -euo pipefail` at top. Validates that all required env vars are set before proceeding: `DATABASE_URL`, `R2_ENDPOINT`, `R2_ACCESS_KEY_ID`, `R2_SECRET_ACCESS_KEY`, `R2_BUCKET_NAME`.

**Configure AWS CLI for R2 in the script** using environment variables:
```
export AWS_ACCESS_KEY_ID="$R2_ACCESS_KEY_ID"
export AWS_SECRET_ACCESS_KEY="$R2_SECRET_ACCESS_KEY"
export AWS_DEFAULT_REGION="auto"
```
Pass `--endpoint-url "$R2_ENDPOINT"` to every `aws s3` / `aws s3api` command.

**`.github/workflows/supabase-backup.yml`** — GitHub Actions workflow:
- `name: Daily Supabase Backup`
- Triggers: `schedule` with cron `0 5 * * *` (5am UTC = 2am Argentina UTC-3). Also add `workflow_dispatch` for manual trigger.
- Single job `backup` running on `ubuntu-latest`.
- Steps:
  1. `actions/checkout@v4`
  2. Install `postgresql-client` (for pg_dump) and `awscli`: `sudo apt-get update && sudo apt-get install -y postgresql-client awscli`
  3. Run the backup script: `bash scripts/supabase-backup.sh`
- Environment variables from GitHub secrets:
  ```yaml
  env:
    DATABASE_URL: ${{ secrets.SUPABASE_DB_URL }}
    R2_ENDPOINT: ${{ secrets.R2_ENDPOINT }}
    R2_ACCESS_KEY_ID: ${{ secrets.R2_ACCESS_KEY_ID }}
    R2_SECRET_ACCESS_KEY: ${{ secrets.R2_SECRET_ACCESS_KEY }}
    R2_BUCKET_NAME: ${{ secrets.R2_BUCKET_NAME }}
  ```

Make the bash script executable with `chmod +x`.
