<objective>
Add pendingSync dirty flag to subgroups, remove the 'sincronizada' estado, wire all mutations to set pendingSync=true, and add syncPending color to theme.

Purpose: This is the data model foundation for the unified bidirectional sync. Without pendingSync, the UI cannot show orange dots and the sync engine cannot track what needs uploading. Removing 'sincronizada' simplifies the state model per D-07.

Output: Drizzle migration (3 files), updated schema, refactored SubGroupRepository, updated queries/hooks that referenced 'sincronizada', theme color token.
</objective>
